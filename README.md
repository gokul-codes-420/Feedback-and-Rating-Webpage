# Feedback and Rating Webpage

A simple from-scratch interactive web application that allows users to submit their feedback and ratings. This project provides an elegant user interface to capture user sentiment and star ratings effectively.

## 🚀 Live Demo

**[Click here to view the live project!](https://gokul-codes-420.github.io/Feedback-and-Rating-Webpage/)**

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **JavaScript**

## 📁 Project Files

- `index.html` - The structural markup for the feedback page (formerly feedback.html).
- `feedback.css` - Styling, layout, and visual enhancements.
- `feedback.js` - Dynamic scripting for processing feedback submissions and interacting with the UI.
